Benvingut/da al 657 Boulevard. Aquesta casa amaga secrets que només podràs descobrir explorant-la amb calma.
Com navegar:

Utilitza el menú lateral dret per moure't entre les diferents seccions: Casa, Cartes, Veïns i El Vigilant.
A la secció Casa, passa el cursor sobre les diferents zones del mapa per descobrir les habitacions.
A la secció Veïns, explora el mapa de cases veïnes fent hover sobre les zones marcades per conèixer qui viu al voltant.
A la secció Cartes, utilitza el menú inferior per seleccionar quina carta vols veure i fes clic sobre la carta gran per obrir-la progressivament: primer apareixerà la carta tancada, després la carta oberta amb el seu contingut.

Pren-te el teu temps per explorar. Cada detall compta. Qui vigila aquesta casa?
